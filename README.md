Donation Management Application :

    This is a full-stack web application designed to facilitate and manage donations efficiently.
    The platform connects donors, associations, and recipients to ensure transparent and organized donation processes.

The application includes a React.js frontend and two backend versions:

     - Spring Boot (Java)
    
     - Laravel (PHP)

It allows users to create and manage donations, while administrators oversee associations, donors, and recipients.
A real-time chat system also enables communication between donors and associations.
